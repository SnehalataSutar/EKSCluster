## Create EKS cluster with 2 nodes using terraform

### Step 1: Prerequisites
Ensure you have:

•	AWS account with IAM permissions for EKS, VPC, IAM, EC2.

•	AWS CLI configured with aws configure.

•	Terraform installed.

•	kubectl installed.
